# Product

**AI-assisted product motion studio** for web & mobile tech — upload real recordings or screenshots, AI drafts markers/scenes, edit in a simple editor, export polished demo videos.

The **editable project** is the product—not a one-shot MP4.

---

## Vision

**One sentence:** Arco turns raw web and mobile app recordings into polished, motion-designed product videos—with AI-suggested zooms, click emphasis, and transitions you customize in a simple editor.

**Category:** Product demo & motion design workflow for web/mobile tech—not generic AI text-to-video, CapCut for everything, or physical product ads.

**Quality bar:** 70% = "cool" → leave. **95% = pay.** Killer demo: raw recording → upload → auto markers → tweak headline → export beats CapCut in 5s side-by-side.

**Strategic insight:** Motion-design productivity for product marketers—not an AI video company. Real competition: AE freelancers, CapCut hours, Screen Studio + manual polish.

## Workflow

```
1. RECORD — browser tab, OBS, QuickTime, iOS/Android screen record
   OR SCREENSHOTS — 3–10 app PNGs/JPGs (storyboard mode)
2. UPLOAD — clip(s) or images to Arco
3. ANALYZE — clicks, pauses, navigation (AI + CV, phased)
4. DRAFT — markers/scenes + zoom / ripple / spotlight / title cards
5. EDIT — sidebar: effects, text, timing, transitions, voice, BGM
6. EXPORT — 16:9, 9:16, 1:1 (+ music, optional VO on screenshots)
```

## Motion presets

AI suggests **where** — presets define **how**:

| Preset | Purpose |
|--------|---------|
| `smooth-zoom` | Focus on UI region |
| `click-ripple` | Emphasize click/tap |
| `spotlight` | Dim surround, highlight area |
| `title-card` | Section headline between beats |
| `feature-callout` | Text + pointer on UI |
| `transition` | Cut, fade, pan between segments |

## Editor (Canva-depth, not AE)

Marker/scene list + sidebar inspector. **Later:** keyframes, layers—only if users demand.
